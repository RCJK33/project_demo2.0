Etiqueta <TABLE>

La etiqueta tabla sirve para organizar, representar datos en 2 o mas dimenciones, sin envargo tambien puede ser usada para divesas cosas como hacer el diseño de un pagina, aunque este ultimo no es recomendable por que se volveria algo muy inisetable y se considera en HTML como un grupo de celdas organizadas en filas y columnas en donde conoceremos mas adelante


Atributo align: Este atributo enumerativo indica como va a encontrarse alineada la tabla respecto al contenido del documento en cuestión, pudiendo tener los siguientes valores:


  • left,significa que la tabla será mostrada a la izquierda del documento;

  • center,significa que la tabla será mostrada al centro del documento;

  • right,significa que la tabla será mostrada a la derecha del documento.

Este metodo de mostrado no debria ser usado debido a que es obsoleto y en su lugar se usara en codigo css las propiedades text-align y vertical-align


Atributo bgcolor, se usa para definir el color de fondo de una tabla o celda haciendo uso de del codigo hexadecimal como es el fotmato sRGB (bajo el prefijo #)

Este metodo tambien es obsoleto y en siu lugar se usara en codigo css backgroud-color


Atributo border Esre atributo define el tamaño de los marcos de las celdas y tabla si este valor equivale a 0 es nulo y no se mostrara nada

Este metodo tambien es obsoleto y poco flexible en su lugar se usara en codigo css. . A fin de dar un efecto similar al atributo, las propiedades CSS nativas border, border-color, border-width y border-style deben ser usadas.


Atributo cellpadding y cellspacing este atributo define el espacio entre su borde y el contenido de la celda, si fuera la longitud de un pixel, dicho espacio se aplicara a cada lado de la celda

Este metodo es obsoleto por lo que en su lugar se recomiendda usar propiedades del codigo CSS en este caso el border-collapse con el valor collapse en el mismo elemento <table>, y la propiedad padding en el elemento <td>.
